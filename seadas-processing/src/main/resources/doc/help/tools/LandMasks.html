Land Masks and Coastline

A shortcut to create land masks for the currently loaded product. The masks are:

  • Water
  • Land
  • Coastline

Location

Tools -> Land Masks and Coastline

Details

This is a convenient shortcut for making masks within a selected product. Normally masks are made one mask at a time and enabled independently for each band. This tool allows the making and configuring of many masks all at one go. Additionally the masks can be set to enabled (visible) in a bands all at once.
