Field Measurements

SeaDAS contains a variety of tools associated with the use of field measurement data.

Field measurement data can be loaded into SeaDAS in one of two formats:
Import SeaBASS File
Import SeaDAS 6.x Track File
Once loaded this data is available for comparative analysis with the satellite data and ship course tracking overlays.



The above image shows a shiptrack for a cruise. The circles are stations where the open circles are the stations which officially matched up based on a 3 hour time window and other criteria. The chlorophyll imagery is from the satellite. The correlation plots show the comparison of the satellite measurement of chlorophyll to that of the field measurement value.

Find Matchups is a tool which returns satellite level-2 file names and download links for a time and point or region.

Make Matchups is a tool which extracts, filters, and appends satellite match-up data to a valid SeaBASS file.
