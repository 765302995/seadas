Map Gridlines

Map Gridlines have been significantly revised for SeaDAS 7.1. Below are some examples. For this release most of the preferences have been removed and decent automated defaults are being used which are dynamic to the image resolution.

The following 4 images were created with SeaDAS using the same Chloryphyll product scene of a level-2 file. They show various applications of map gridlines along with the effects of reprojection on the map gridlines.

  • Image 1 shows map gridlines on an image of standard l2-file
  • Image 2 shows map gridlines on a reprojection of the same file as Image 1
  • Image 3 shows another variant of Image 2, but with labels placed inside the image
  • Image 4 shows another variant of Image 1, but with only corner labels and no gridlines

To edit any of the parameters click on the Layer Editors icon . This will enable you to edit whichever layer is currently selected. If the Map Gridline layer does not show up in the Layer Editors window, then toggle the Map Gridlines button off then on, this will make the Map Gridline layer the selected layer. The Layer Manager window is another place where you may select a layer to be the selected layer.

The following image shows the user editable components of the map gridlines. Most fields are pretty self-explanatory. The latitude and longitude step sizes may be set directly where the value represent degrees between the gridlines. Setting this field to zero results in Auto-Sizing. The Auto-Sizing algorithm will set the gridline spacing to a reasonable value such that there are about 3-5 lines. Also note that lines are placed at the nearest most sensible degree value.
