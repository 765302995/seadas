Map Gridlines

Map Gridlines have been significantly revised for SeaDAS 7.1. Below are some examples. For this release most of the preferences have been removed and decent automated defaults are being used which are dynamic to the image resolution.

To edit any of the parameters click on the Layer Editors icon . This will enable you to edit whichever layer is currently selected. If the Map Gridline layer is does not show up in the Layer Editors window, then toggle the Map Gridlines button off then on, this will make the Map Gridline layer the selected layer. The Layer Manager window is another place where you may select a layer to be the selected layer.
