SeaDAS 7.1.1 Release Notes

Release Date: March 27, 2015

New Features

        Bathymetry
            - Depth values are now in positive units.
            - Default color scheme is in log space to better depict shallow water floor features.

        Color Palettes Added/Revised
            - Bathymetry color palette revised with a smoother transition from light blue-green to dark blue-green.
            - Various gradient palettes added.
            - Default (oceancolor_standard.cpd) revised to not use black at high saturation.

        Color Schemes:
            - Added a user defined text file for generating custom schemes or overriding default schemes.
            - Added a wildcard feature to enable more generic auto-application of color schemes.

        Band Filters Added
            - Low-Pass Radial Attenuation
            - Straylight

        RGB Profile Added
            - MODIS Trucolor profile added.

Bug Fixes

        Color Schemes
