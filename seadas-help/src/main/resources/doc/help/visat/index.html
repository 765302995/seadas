SeaDAS 7.1 Component Guide

File Tools

File Manager

Displays all opened files in a tree view.
Tools -> Tool Windows -> File Manager   (Docked: left side)

Image Tools

Color Manager

Set the color palette of band image.

Ground Control Points

Manage ground control points.

Layer Editor

Edit layer properties (Grid Lines, No-Data, etc.).

Layer Manager

Manage view layer visibility and transparency.

Mask Manager

Manage binary data masks and ROIs (regions of interest).

Navigation

Zoom and pan a selected image view.

Pin Manager

Manage placement of pins.

World Map

Display file geographic boundaries on a world map.

Analysis Tools

Correlative Plot

Displays Correlative Plot for a selected band.

Histogram Plot

Displays Histogram for a selected band.

Profile Plot

Displays Profile Plot for a selected band.

Scatter Plot

Displays Scatter Plot for two selected bands.

Spectrum Plot

Display spectra at the current pixel position and for pins.

Statistics

Displays Statistics for a selected band.

Info Tools

Band Info

Display information for a selected band.

Geo-Coding View

Display Geo-Coding for a selected band.

Pixel Info

Displays detailed pixel information.
Info -> Pixel Info   (Docked: right side)

Tile Cache Diagnosis

Vector Data Info

Displays attributes of selected vector data features.
Info -> Vector Data Info
