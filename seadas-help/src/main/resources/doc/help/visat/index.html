SeaDAS 7.1 Component Guide

File Tools

File Manager

Displays all opened files in a tree view.
Tools -> Tool Windows -> File Manager   (Docked: left side)

General Processing Tools

Collocate

Collocate two files into a single file (useful for parameter comparisons, etc.

Crop

Crop a file spatially (default boundaries are the current view).

Flip Data

Apply Data Flip.

Mosaic

Create a mosaic of multiple files.

Orthorectify

Apply an orthorectification.

Reproject

Apply a reprojection.

Band Tools

DEM Band

Creates DEM elevation and optionally DEM-corrected lat/lon bands from a selected DEM.

Filtered Band

Creates a filtered band from the currently selected band.

Math Band

Create a new band using an arbitrary mathematical expression.

Real Band

Computes a "real" band from a virtual or filtered band.

Image, Layer, & Mask Tools

Bathymetry

Add elevation, bathymetry, and topography bands and bathymetry mask.

Coastline, Land & Water

Add coastline, land and water masks

Colorbar

View and/or Export the colorbar.

Color Manager

Set the color palette of band image.

Contour

Add contour lines.

Gridlines

Show/hide map gridlines layer for the selected image.

Layer Editor

Edit layer properties (Grid Lines, No-Data, etc.).

Layer Manager

Manage view layer visibility and transparency.

Mask Manager

Manage binary data masks and ROIs (regions of interest).

Navigation

Zoom and pan a selected image view.

No-Data Layer

Show no-data overlay for the selected image.

World Map Layer

Show World Map layer for the selected image.

World Map Location

Display file geographic boundaries on a world map.

Pin Tools

Pin Manager

Manage placement of pins.

Pin Placer

Places pins via mouse click onto image.

Geometry Tools

Draw Rectangle

Draw Ellipse

Draw Polygon

New Vector Container

Create a new vector data container for drawing lines and polygons.

New (Text-Based) Geometry

Enter coordinates to create a geometry (well-known-text format)

Geometry Info

Get the well-known-text (WKT) representation of a selected geometry.

Export Geometry

Export Geometry as Shapefile.
File -> Vector Export -> Shapefile

Import Shapefile

Import geometry from ESRI Shapefile.
File -> Vector Import -> Shapefile

Ground Control Point Tools

Ground Control Points

Manage ground control points.

Ground Control Points Placer

Places ground control points via mouse click onto image.

Analysis Tools

Correlative Plot

Displays Correlative Plot for a selected band.

Histogram Plot

Displays Histogram for a selected band.

Profile Plot

Displays Profile Plot for a selected band.

Scatter Plot

Displays Scatter Plot for two selected bands.

Spectrum Plot

Display spectra at the current pixel position and for pins.

Statistics

Displays Statistics for a selected band.

Info Tools

Band Info

Display information for a selected band.

Geo-Coding

Display Geo-Coding for a selected band.

Pixel Info

Displays detailed pixel information.
Info -> Pixel Info   (Docked: right side)

Tile Cache Diagnosis

Vector Data Info

Displays attributes of selected vector data features.
Info -> Vector Data Info
