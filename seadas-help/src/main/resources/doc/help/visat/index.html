Help

SeaDAS is a scientific data visualisation, analysis, and processing software application for remote-sensing satellite data which derives from and is an extension of the BEAM Earth Observation Toolbox and Development Platform. SeaDAS includes support for the data products produced by NASA's Ocean Biology Processing Group (http://oceancolor.gsfc.nasa.gov). This support includes data processing capabilities to allow users to process data between the various levels (i.e. Level 0 through Level 3).

Please be aware that some of the component-level SeaDAS 7.1 help pages have not yet been fully revised to match the current version. For the latest SeaDAS help and tutorials see http://seadas.gsfc.nasa.gov

Collocate

Creates a new file which is a collocation of two selected files. (useful for parameter comparisons, etc). The Collocation Tool allows to collocate two spatially overlapping products. Collocating two products implies that the pixel values of one product (the dependent) are resampled into the geographical raster of the other (the reference).

Processing -> Collocate
Toolbar: Processing

DEM Band

Creates a DEM (Digital Elevation Model) band and optionally DEM-corrected lat/lon bands from a selected DEM.

Tools -> DEM Band

Extract Pixel Values

Extracts the pixel values given by a user-specified list of geographic coordinates from a given list of data files.

Tools -> Extract Pixel Values

Filtered Band

Creates a filtered band from the currently selected band. Filtered bands are created by applying convolution or non-linear filters to the selected band. Filters are used to perform common image processing operations e.g. sharpening, blurring or edge enhancement.

Tools -> Filtered Band
Toolbar: Band Tools

Math Band

Create a new band using an arbitrary mathematical expression. The band maths tool is used to create new image sample values derived from existing bands, tie-point grids and flags. The source data can originate from all currently open and spatially compatible input products. The source data is combined by arbitrary mathematical expression to generate the target data.

Tools -> Math Band
Toolbar: Band Tools

Real Band

Computes a "real" band from a virtual or filtered band.

Tools -> Real Band

Principle Component

Principal Component Analysis (PCA) is typically used to develop a linear model for a set of data (e.g. face images) which can then be used to test for membership. PCA works by converting the set of data to a new basis that is a subspace of the original set. The subspace is selected to maximize information.
