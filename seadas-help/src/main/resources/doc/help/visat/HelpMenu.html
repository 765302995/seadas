Help Menu

Note: this help page is for SeaDAS 7.0. The current release, SeaDAS 7.1, has a major redesign to it's menu structure and naming scheme, as well as new features and alterations. Even though this page is out-of-date, much of it's information and links can be relevant and useful. This page will be updated with the next SeaDAS release.


SeaDAS's Help Menu

The SeaDAS Help Menu contains:

  •   Help Topics
    Opens the SeaDAS help.
  • SeaDAS Home Page
    Opens the SeaDAS web site. http://seadas.gsfc.nasa.gov
  • Access to Data Products
    Shows an information page of data product distributors.
  •   Module Manager...
    The Module Manager for updating, installing and unstalling modules.
  • Check for New Release...
    Checks for a new BEAM release.
  •   About SeaDAS
    Opens the 'About SeaDAS' window.
