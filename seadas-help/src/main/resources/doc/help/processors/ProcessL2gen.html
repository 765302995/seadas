Data Processing Tools

l2gen

Creates a Level 2 file from an input Level 1 (A or B) file.

Location

Tools -> OCSSW Processing -> l2gen

Details

When the l2gen GUI is opened if their is currently a product loaded in SeaDAS then it's input file will be loaded into the ifile field. Otherwise the majority of the tabs and fields will be greyed out awaiting the selection of an ifile. The ifile may be selected using the selector or it can be loaded in via a parfile. Once an ifile has been determined, the GUI will be populated based on the ifile. Most of the tabs as self explanatory, the Products tab is more involved see: Specification of l2gen Products.
See also l2gen -help
