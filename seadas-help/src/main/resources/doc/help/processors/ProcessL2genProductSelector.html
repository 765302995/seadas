Specification of l2gen Products

The l2gen product parameter 'l2prod' is initially set to the default value. At any time l2prod can be restored to the default value by clicking 'Apply Defaults' on the L2gen:Products tab). l2prod can be specified in a multitude of ways. Note that a change within any of one these product specifier controls will automatically update the others.

  • The 'Product Selector' tree (located within the l2gen:Products tab)
  • The 'Selected Products' textfield (located within the l2gen:Products tab)
  • The 'Parameters' textfield (located within the l2gen:Overview tab)
  • A 'Parameter File' (which can be uploaded within the l2gen:Overview tab)

Product Selector

This selector is located within the l2gen:Products tab. The products are grouped into logical categories within this tree-based selector. Any branch may be fully unwound and an individual product may then be selected. Selection at any parent level will result in the selection of all the child level products. By default checking/unchecking a parent behaves in a dual state manner, respectfully checking/unchecking all of it's children. For selecting products which have a multitude of wavelength options, the 'Wavelength Limiter' tool is available (located at the top right of the same tab). This tool is especially useful when a partial wavelength selection is desired. If the 'Wavelength Limiter' is partially selected then the 'Product Selector' checkboxes for the parent level take on a tri-state behaviour of (none, partial, all), where partial reflects the selected criteria within the 'Wavelength Limiter'. Note that a change in the 'Wavelength Limiter' will not directly change the products selected, but only effects the behaviour of the 'Product Selector'.

Selected Products

This textfield located within the L2gen:Products tab displays the current selection of 'l2prod'. This field is editable: click 'Edit', then once edits are complete click 'Load'. Note that until 'Load' is clicked the edits are not final and do not appear in the other controls. Clicking 'Cancel' during the edit will restore the values prior to the edit.
